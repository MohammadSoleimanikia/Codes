add items to menu

  1. add a div with these class: collapse navbar-collapse
  2. add another div inside of that div with class of: navbar-nav
  3. inside of navbar-nav div add link items with class of nav-link
  4. add navbar-expand-(sm,lg,xl) to declare when menu expanded
  5. add button for when menu collapsed with class of navbar-toggler
  6. add data-bs-target with value equal to div with collapse navbar-collapse class
  7. add span with navbar-toggler-icon class inside of button to add hamburger icon
